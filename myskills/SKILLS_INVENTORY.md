# Skill 完整清单

## 🚀 新产品经理技能 (New Product Manager Skills)
这些是最新开发或增强的 PM 核心工具，具备高度的 AI 赋能和标准设计规范。

| Skill 名称 | 路径 | 核心功能描述 |
| :--- | :--- | :--- |
| **ai-opportunity-map-generator** | `new-product-manager-skills/ai-opportunity-map-generator` | 根据业务流程描述生成符合标准设计规范的“AI机会场景地图” HTML文件，用于挖掘自动化和智能化机会。 |
| **applying-brand-guidelines** | `new-product-manager-skills/applying-brand-guidelines` | 为所有生成的文档应用一致的企业品牌和样式，包括颜色、字体、布局和消息传递。 |
| **prototype-prompt-generator** | `new-product-manager-skills/prototype-prompt-generator` | 基于产品想法或需求，生成高质量的 UI/UX 原型初始化提示词，适用于 v0、Claude Artifacts 等工具。 |

## 📈 商业与职场研究 (Business Skills)
专注于市场调研、财务分析及职场薪酬数据。

| Skill 名称 | 路径 | 核心功能描述 |
| :--- | :--- | :--- |
| **job-market-analyzer** | `business-skills/job-market-analyzer` | **宏技能**：调研各大厂（大厂、科技公司）岗位职级薪资（TC/Base/年终/期权）。支持 Liepin、脉脉大爆料数据。 |
| ├── data-collection | `.../sub-skills/data-collection` | 子技能：采集 Levels.fyi、脉脉、猎聘等平台的原始薪酬数据。 |
| ├── data-cleaning | `.../sub-skills/data-cleaning` | 子技能：清洗和翻译原始薪资数据，统一映射为标准化 JSON。 |
| ├── data-analysis | `.../sub-skills/data-analysis` | 子技能：对清洗后的数据进行统计分析，产出图表数据。 |
| └── report-generation | `.../sub-skills/report-generation` | 子技能：基于统计结果生成最终调研报告。 |
| **analyzing-financial-statements** | `business-skills/analyzing-financial-statements` | 计算财报数据中的关键财务比率和指标，用于投资分析。 |
| **creating-financial-models** | `business-skills/creating-financial-models` | 提供高级财务建模套件，包括 DCF 分析、敏感性测试、蒙特卡洛模拟等。 |
| **data-visualizer-pro** | `business-skills/data-visualizer-pro` | 将结构化数据（CSV/JSON）转化为包含交互图表的专业 HTML 数据报表。 |
| **deep-research-electronics** | `business-skills/deep-research-electronics` | 深入调研消费电子巨头（苹果、三星、小米等）的 AI 举措，生成一页纸 PPT 趋势报告。 |

## 🛠️ 产品经理方法论 (Product Manager Skills)
包含 46 个涵盖从战略到执行的全流程 PM 框架。

### 核心战略与定位 (Strategy & Positioning)
| Skill 名称 | 核心功能描述 |
| :--- | :--- |
| **product-strategy-session** | 运行端到端产品战略会议，涵盖定位、发现和路线图规划。 |
| **positioning-workshop** | 运行定位工作坊，明确目标客户、需求、类别、收益和差异化。 |
| **positioning-statement** | 创建 Geoffrey Moore 风格的定位声明。 |
| **pestel-analysis** | 宏观环境分析工具。 |
| **tam-sam-som-calculator** | 市场规模计算工具。 |

### 需求发现与流程 (Discovery & Process)
| Skill 名称 | 核心功能描述 |
| :--- | :--- |
| **discovery-process** | 运行从问题假设到验证方案的全周期发现流程。 |
| **discovery-interview-prep** | 准备用户调研访谈。 |
| **problem-framing-canvas** | 业务问题定义与框架构建。 |
| **problem-statement** | 定义清晰的问题陈述。 |
| **opportunity-solution-tree** | 机会解决方案树，通过实验验证想法。 |
| **lean-ux-canvas** | 框架化业务问题、假设及后续学习。 |
| **jobs-to-be-done** | 结构化 JTBD 格式，识别用户未满足的需求。 |
| **proto-persona** | 基于现有研究创建原型人格。 |

### 产品定义与路线图 (Definition & Roadmap)
| Skill 名称 | 核心功能描述 |
| :--- | :--- |
| **roadmap-planning** | 规划战略路线图，涵盖优先级、史诗定义和利益相关者对齐。 |
| **epic-hypothesis** | 将史诗框架化为可测试的假设。 |
| **epic-breakdown-advisor** | 根据 Humanizing Work 模式将大型史诗拆解为用户故事。 |
| **prd-development** | PRD 文档撰写与迭代。 |
| **user-story-mapping-workshop** | 运行用户故事地图工作坊，规划发布切片。 |
| **user-story** | 使用 Mike Cohn 格式和 Gherkin 标准编写用户故事。 |
| **user-story-splitting** | 将过大的故事拆分为可交付的小项。 |
| **prioritization-advisor** | 根据场景选择 RICE, ICE 等权重框架。 |

### 商业运营与分析 (Business & Metrics)
| Skill 名称 | 核心功能描述 |
| :--- | :--- |
| **saas-revenue-growth-metrics** | 计算 SaaS 收入、留存和增长指标，诊断产品市场匹配度。 |
| **saas-economics-efficiency-metrics** | 评估 SaaS 单元经济（LTV/CAC）和资本效率。 |
| **finance-metrics-quickref** | 快速查阅 SaaS 财务指标定义、公式和基准。 |
| **finance-based-pricing-advisor** | 使用 ARPU、NRR 等指标评估调价风险。 |
| **feature-investment-advisor** | 基于营收、成本和 ROI 评估功能投资价值。 |
| **business-health-diagnostic** | 业务健康状况全盘诊断。 |

### 用户体验与反馈 (UX & Feedbacks)
| Skill 名称 | 核心功能描述 |
| :--- | :--- |
| **customer-journey-map-workshop** | 运行包含阶段、情绪和机会点的用户旅程制图工作坊。 |
| **customer-journey-map** | 创建端到端的用户旅程地图。 |
| **storyboard** | 通过故事板可视化用户场景。 |
| **press-release** | 亚马逊风格的“由终向始”新闻稿，定义客户价值。 |
| **eol-message** | 编写富有同理心的产品下线/退市通知。 |

| **vp-cpo-readiness-advisor** | 指导向 VP 或 CPO 级别跃迁的高级顾问。 |
